# Coding Interview Questions

A collection of practical coding interview questions with runnable starter examples.

## Questions

1. [DOM Manipulation: Dynamic Table From JSON](questions/dom-dynamic-table/README.md)
2. [Algorithm: Check If Two Strings Are Anagrams](questions/anagram-check/README.md)
3. [Data Structures: Validate HTML-like String Nesting](questions/html-nesting-validator/README.md)

## How to Run

Each question lives in its own folder.

TypeScript solutions can be built from the repository root:

```powershell
npm.cmd run build
```

For browser-based questions, build first and then open the `index.html` file in a browser.
