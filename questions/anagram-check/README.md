# Algorithm: Check If Two Strings Are Anagrams

## Problem Statement

Write a function that checks whether two strings are anagrams.

Two strings are anagrams if they contain the same characters with the same frequency, but possibly in a different order.

## Examples

```text
Input: "listen", "silent"
Output: true
```

```text
Input: "hello", "world"
Output: false
```

```text
Input: "Dormitory", "Dirty room"
Output: true
```

## Requirements

- Ignore spaces.
- Ignore letter casing.
- Return `true` if both strings are anagrams.
- Return `false` otherwise.

## Bonus

- Handle punctuation gracefully.
- Solve it using a character frequency map instead of sorting.
- Mention the time and space complexity.

## Run

Use Node.js:

```powershell
node solution.js
```
